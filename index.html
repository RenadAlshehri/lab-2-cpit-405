Lab 2: HTML

CPIT-405 GitLab Page

Welcome to the CPIT-405 lab session. In this lab, we will explore the basics of HTML and learn how to structure a web page using various HTML tags. Understanding these fundamentals is crucial for web development and creating dynamic and interactive web pages.

HTML stands for HyperText Markup Language, and it is the standard language used to create web pages. It allows developers to structure content on the web and define elements such as headings, paragraphs, links, images, tables, and forms. Mastering HTML is the first step towards becoming a proficient web developer.
